layui基础使用

by 菲尼莫斯 2018年7月17日

下载layui

  1. 下载layui

  2. 将整个插件放入static文件下

引入layui

  1. 在.html文件的head元素下加入 <link rel=“stylesheet” href="/layui/css/layui.css">

  2. 在.html文件末加入 <script charset=“UTF-8” src="/layui/layui.js"></script>

  var form = layui.form

  //初始化日期选择器

  });
