**npm ls xxx** : 查看当前安装模块的版本

**npm list -g --depth 0** : 查看当前安装的所有全局模块

**npm install xxx@1.11** : 安装指定的版本，会覆盖原来的版本

**npm uninstall xxx** : 卸载模块

**npm install xxx@^4.19.1** ：匹配最新的4.xx.xx版本

**npm install xxx@~4.19.1** ：匹配最新的4.19.xx版本
